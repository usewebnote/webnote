How to Save a file in WebNote

To save a file in WebNote just simply scroll to the bottom of the page until you see the save button

Click on it and you will be prompted to name the file (Please make sure it has the .txt extension e.g file.txt) and specify the loctaion of the file in your storage

How to open a file in WebNote

To open a file in WebNote just simply scroll to the bottom of the page until you see the Choose File button

Click on it and you will be prompted to choose your file
